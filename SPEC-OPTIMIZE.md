# TaskFlow 优化任务清单

> **版本**: 1.0.0
> **更新**: 2026-03-04

## 概述

修复任务列表和任务树页面的问题，提升用户体验。

---

## 1. 任务列表页优化

### 1.1 进度条问题

**问题**: 进度条长度不一，数字大了会挤占进度条

**修复方案**:
- 进度条使用固定宽度（如 80px），不随数字变化
- 进度数字单独显示在进度条右侧
- 示例: `[======>     ] 75%`

### 1.2 筛选功能修复

**问题**: 筛选优先级/标签无法使用，缺少筛选项目

**修复方案**:
- 筛选优先级：使用 NSelect，下拉选择高/中/低
- 筛选标签：使用 NSelect，多选标签
- 新增筛选项目：使用 NSelect，下拉选择项目（支持"全部"选项）

### 1.3 筛选 UI 布局

```
[筛选状态] [筛选优先级] [筛选标签] [筛选项目] [刷新]
```

## 2. 任务树页面重构

### 2.1 界面全新设计

**需求**:
- 按"项目 -> 任务 -> 子任务 -> 子子任务..." 层级组织
- 使用树形组件展示（如 NTree 或自定义递归组件）
- 每个项目展开显示其下的任务树

### 2.2 数据结构

```typescript
interface TreeNode {
  id: number
  title: string
  type: 'project' | 'task'
  children?: TreeNode[]
  status?: string
  is_blocked?: boolean
}
```

### 2.3 交互设计

- 点击项目名可展开/折叠任务列表
- 点击任务条目在**右侧抽屉**显示详情（不是跳转页面）
- 使用 NDrawer 实现右侧抽屉
- 抽屉内显示任务详情，可编辑

### 2.4 样式设计

- 项目名称：加粗，左侧显示项目图标
- 任务：缩进显示，状态用颜色标识
- 子任务：递归缩进，每级缩进 20px
- 阻塞状态：灰色显示 + 锁图标
