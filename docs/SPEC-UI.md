# AITaskFlow 界面优化方案

## 一、布局调整

### 当前问题
```
┌────────┬──────────────────────┐
│        │  Header（只在这栏）  │
│ 左侧菜单├──────────────────────┤
│        │  内容区              │
└────────┴──────────────────────┘
```

### 调整后
```
┌──────────────────────────────────┐
│  Header（全宽通栏，Logo+搜索）   │  ← 顶层
├────────┬─────────────────────────┤
│ 左侧菜单│  内容区               │
└────────┴─────────────────────────┘
```

## 二、风格迭代

### 设计方向：专业、简洁、现代

#### 1. 配色方案
- **主色**：深蓝色 #1a56db（专业感）
- **辅色**：浅蓝 #dbeafe（背景、标签）
- **强调色**：橙色 #f97316（按钮、提醒）
- **背景**：浅灰 #f8fafc（内容区）
- **文字**：深灰 #1e293b

#### 2. 左侧菜单
- 深色背景 #1e293b（接近黑色）
- 白色图标+文字
- 选中项：主色高亮条

#### 3. Header
- 白色背景 + 底部细阴影
- Logo 更精致
- 搜索框圆角 + 阴影

#### 4. 整体视觉
- 卡片式设计（内容区）
- 圆角 8-12px
- 柔和阴影
- 间距宽松

## 四、实现要点

1. n-layout 结构调整：外层加 header
2. 左侧菜单：深色主题
3. 内容区：白色/浅灰背景
4. 卡片：圆角 + 阴影
5. 按钮：主色渐变或实色
