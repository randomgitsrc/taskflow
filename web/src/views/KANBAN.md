# 看板视图设计方案

## 概述
在"任务列表"页面增加看板视图模式，类似于 Trello，按状态列展示任务卡片，支持拖拽切换状态。

## 页面布局


## 功能设计

### 1. 视图切换
- 在任务列表页面顶部增加切换按钮：[列表视图] [看板视图]
- 默认列表视图，看板视图可选
- 切换时保持筛选条件

### 2. 看板列
- 每列对应一个任务状态：待处理、进行中、已完成、已暂停、阻塞、等待中
- 列头显示状态名 + 任务数量
- 可折叠/展开列（可选）

### 3. 任务卡片
- 显示内容：标题、优先级标签、项目名、进度条
- 鼠标悬停显示详情（描述、负责人等）
- 点击可编辑任务

### 4. 拖拽功能
- 使用 vuedraggable 或 html5 drag API
- 拖拽任务到其他列 → 自动更新状态
- 拖拽排序（同列内调整顺序）

### 5. 新建任务
- 每个列底部有 [+新建] 按钮
- 点击直接在对应状态创建任务

## 技术实现

### 依赖
- vuedraggable（@vuedraggable/core）或 @vueuse/core 的 useDraggable

### 数据结构
```typescript
interface KanbanColumn {
  status: string
  label: string
  tasks: Task[]
}
```

### 组件结构
```
TaskList.vue
  ├── 视图切换按钮
  ├── 筛选栏
  ├── [列表模式] n-data-table
  └── [看板模式] KanbanBoard.vue
        └── KanbanColumn[]
              └── TaskCard[]
```
